Welcome to Little Lemon

A family-owned Mediterranean restaurant focused on traditional recipes served with a modern twist.

API Endpoints

  • POST /api/registration/ — Register a new user
  • POST /api/token/ — Get auth token
  • GET/POST /api/menu/ — List or create menu items
  • GET/PUT/DELETE /api/menu/<id>/ — Single menu item
  • GET/POST /api/bookings/ — List or create bookings
  • GET/PUT/DELETE /api/bookings/<id>/ — Single booking
